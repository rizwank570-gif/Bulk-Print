🖨️ Auto-Print & Stapling Setting Kaise Karein?

Step 1: Printer me Staple Set Karna (Ek Baar)

Kyunki Auto-Print jaldi me print karta hai, aapko pehle se default setting set karni hogi:

  • Apne PC me Printers & Scanners search karke open karein.
  • Apne Printer par click karke Printing preferences kholein.
  • Wahan apni setting set karein (Jaise: Staple ON, A4 Paper, wagaira).
  • Apply aur OK kar dein. Ab apka printer hamesha staple karega!

Step 2: Chrome me Auto-Print Chalu Karna

Saari files bina dialog ke lagatar print karne ke liye:

  1. Desktop par Google Chrome ke icon par Right-Click karein > Properties dabayein.
  2. Target wale box me sabse aakhir mein ek space (khaali jagah) dekar yeh likhein:
"C:\Program Files\Google\Chrome\Application\chrome.exe" --kiosk-printing
Dhyan De: .exe" aur --kiosk ke beech mein SPACE hona bahut zaroori hai. Change save karne ke baad sabhi khule hue Chrome windows ko pehle band kar dein, fir is nayi shortcut se tool open karein.
